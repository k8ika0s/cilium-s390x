# s390x Log Archive

Raw run logs for s390x validation are stored under UTC-date directories:

- `docs/s390x/logs/YYYY-MM-DD/`

## Rules

- Do not edit historical `.log` evidence files.
- Add a `RUN-INDEX.md` file per date to summarize outcomes.
- Keep log naming consistent with `docs/s390x/reporting-flow.md`.
- Treat this tree as local evidence storage for remediation and upstream prep.
